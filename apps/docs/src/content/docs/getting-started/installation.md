---
title: Installation
description: Install Minions Skills packages
---

## TypeScript / Node.js

```bash
npm install @minions-skills/sdk minions-sdk
```

## Python

```bash
pip install minions-skills
```

## CLI

```bash
npm install -g @minions-skills/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
