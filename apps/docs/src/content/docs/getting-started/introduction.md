# Minions Skills

Reusable skill definitions that agents can load, compose, and version

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-skills/sdk`) — Core library for Node.js
- **Python SDK** (`minions-skills`) — Python bindings
- **CLI Tool** (`@minions-skills/cli`) — Command-line interface
- **Web App** — Interactive playground at [skills.minions.wtf](https://skills.minions.wtf)
